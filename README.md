# Gonzaga Art & Shine — Restyling

> **"Elegância que nasce da terra"**

Repositório de trabalho activo para o restyling completo da marca **Gonzaga Art & Shine** (Hugo Gonzaga Gomes) — joalharia alternativa mid-range, posicionada para festivais, raves e artesanato exclusivo.

---

## Stack Técnica

- **Backend:** Node.js 20 LTS + Express 4 + EJS templates
- **Base de dados:** MySQL 8
- **Servidor:** Ubuntu VPS — Nginx + PM2 + Certbot (Let's Encrypt)
- **CI/CD:** GitHub Actions → SSH deploy
- **Pagamentos:** Stripe
- **Auth clientes:** Passport.js (local strategy)
- **Emails:** Resend.com
- **Carousel:** Swiper.js
- **Fontes:** Cinzel (títulos) + Source Sans 3 (corpo) — Google Fonts
