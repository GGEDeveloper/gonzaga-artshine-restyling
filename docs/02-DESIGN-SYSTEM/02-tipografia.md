# Tipografia

## Famílias

| Família | Fonte | Uso |
|---|---|---|
| Títulos | Cinzel | H1, H2, logótipo, slogan, nomes de colecção |
| Corpo | Source Sans 3 | Parágrafos, labels, navegação, preços |

**Origem:** Google Fonts (gratuito, sem custo externo)

## Hierarquia Tipográfica

```
H1  — Cinzel, 48–72px, letter-spacing: 0.08em, uppercase
H2  — Cinzel, 32–40px, letter-spacing: 0.06em
H3  — Cinzel, 24px, letter-spacing: 0.04em
Body — Source Sans 3, 16–18px, line-height: 1.6
Small — Source Sans 3, 13–14px
Label — Source Sans 3, 11px, uppercase, letter-spacing: 0.1em
```

## Regras de Uso

- Cinzel é **exclusivo** para títulos e elementos de marca — nunca em blocos de texto
- Source Sans 3 em toda a UI funcional (botões, forms, navegação)
- Cor de texto padrão: `#C7CACE` (prata fosca) em fundo escuro
- Nunca usar branco puro (`#FFFFFF`) — usar sempre a prata fosca

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Cinzel:wght@400;600;700&family=Source+Sans+3:wght@300;400;600&display=swap" rel="stylesheet">
