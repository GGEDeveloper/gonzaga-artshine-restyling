# Componentes UI

> Estado inicial — a detalhar durante a fase de análise de estilos.

## Botões

| Tipo | Fundo | Texto | Border |
|---|---|---|---|
| Primário | `#B08D57` | `#0B0D0C` | none |
| Secundário | transparent | `#C7CACE` | 1px `#C7CACE` |
| Ghost | transparent | `#B08D57` | 1px `#B08D57` |
| Danger | `#5C1A1A` | `#C7CACE` | none |

## Cards de Produto

- Fundo: `#0F2E20` (verde musgo) ou `#3A2C22` (terra)
- Overlay: gradiente de baixo para cima
- Hover: escala `1.02` + sombra dourada subtil
- Badge família: prata ou dourado conforme regra 50/50

## Navegação

- Fundo header: `#0B0D0C` com `backdrop-filter: blur(8px)`
- Links: Source Sans 3, uppercase, letter-spacing
- Active state: underline dourado

## Formulários

- Input border: `1px solid #3A2C22`
- Input focus: border `#B08D57`
- Background inputs: `rgba(255,255,255,0.04)`
