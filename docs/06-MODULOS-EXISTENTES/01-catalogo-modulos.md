|---|---|---|

| 3 | Quick View | Modal de pré-visualização do produto |

## Regras

- Qualquer restyling deve manter a API pública destes módulos
- Podem ser re-estilizados (CSS) mas não refactored (JS)
- Testar todos após qualquer alteração de HTML no catálogo
